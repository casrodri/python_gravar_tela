# Gravador de Tela em Python

Este script em Python utiliza as bibliotecas OpenCV, NumPy, PyAutoGUI e Keyboard para gravar a tela do computador e salvar o vídeo resultante em um arquivo AVI.

## Pré-requisitos

- Python 3.x
- OpenCV (`pip install opencv-python`)
- NumPy (`pip install numpy`)
- PyAutoGUI (`pip install pyautogui`)
- Keyboard (`pip install keyboard`)

## Como Usar

1. Clone o repositório ou faça o download do script.

2. Instale as bibliotecas necessárias utilizando os comandos acima.

3. Execute o script usando um interpretador Python.

4. O script começará a gravar a tela. Pressione a tecla 'Esc' para encerrar a gravação.

5. O vídeo resultante será salvo como "gravacao.avi" no diretório do script.

## Configurações

- O codec utilizado é XVID, e a taxa de frames é fixada em 30 frames por segundo (pode ser ajustado conforme necessário).
- O nome do arquivo de saída pode ser modificado no código.
